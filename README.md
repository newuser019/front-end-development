# front-end-development
心胸围术期肺康复护理数据管理平台前端开发
技术栈：HTML / CSS / JavaScript / HTTP / 数据接口对接
负责数据管理平台的前端全流程开发，实现患者信息管理、康复数据监测、多维度筛选 / 分页 / 导出等核心功能，为医护业务场景提供稳定、直观的数据服务支持
基于原生 JS 封装通用表格、弹窗、表单校验等业务组件，兼容边界场景处理，具备 SDK 级模块化开发与高质量代码交付能力
针对大数据量表格渲染与交互性能进行优化，实现懒加载、虚拟滚动与批量操作逻辑，降低首屏加载耗时，提升系统稳定性与用户体验
完成前后端接口对接，处理 HTTP 请求异常、数据格式适配与错误重试逻辑，保障数据传输的可靠性，具备端到端的全栈开发实践能力
